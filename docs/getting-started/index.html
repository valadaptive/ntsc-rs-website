Getting Started

ntsc-rs is available as a standalone application, an OpenFX plugin, and an After Effects / Premiere plugin.

Downloading

You can download the latest version of ntsc-rs from the GitHub releases section.

Download the correct version for your platform and whichever type you prefer (standalone or OpenFX).

Currently, only Windows and Linux are supported out of the box. macOS support requires you to build it yourself, but it should function.

Installation

Follow the instructions for:
  • the standalone application.
  • the OpenFX plugin.
  • the After Effects / Premiere plugin.
